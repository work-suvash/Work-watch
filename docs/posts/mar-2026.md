# Stars Added ⭐

- Starred both [Epstein Exposed + JMail](https://workwatch.net/reading#declassified-leaked-documents) in documents section, we've been sent a lot of these, but these 2 seem to have some of the better UIs.

- Starred [Convert to it!](https://workwatch.net/file-tools#file-converters) in File Converters, nearly all formats imaginable, file uploads are in browser only, not uploaded to them which is good for privacy.

- Starred [ZeroFS](https://workwatch.net/file-tools#file-hosts) in File Hosting, unlimited file size, keeps files for 30 days after its last download.

- Starred [GN-Math](https://workwatch.net/gaming#unblocked-games) in Unblocked Browser Games, has a bunch of good ports, seems to be well liked.

- Starred [Font Analyzer](https://workwatch.net/text-tools#font-tools) in Font Tools, analyses fonts from any page and gives you similar alternatives.

- Starred [Markbox](https://workwatch.net/text-tools#pastebins) (aka fluffle) in Pastebins, alt to rentry with similar UI, has markdown support, editing, etc.

- Starred [Efecto](https://workwatch.net/image-tools#image-effects) in Image Editing, web-based visual effects editor, supports images, videos and 3D models.

- Starred [KLIPY](https://workwatch.net/ai#video-generation) in Video Gen, allows you to generate unlimited GIFs using Veo 3.

- Starred [Image-Upscaling](https://workwatch.net/image-tools#upscale-restore), online image upscaling tool, works quickly, gives good results.

- Starred [Google Flow](https://workwatch.net/ai#image-generation) in Image Gen, gives unlimited Nano Banana Pro / Nano Banana 2. 

- Starred [Khoj](https://workwatch.net/ai#image-generation) in Image Gen, gives unlimited Nano Banana / Imagen 4.

- Starred [Knots 3D](https://workwatch.net/educational#skills-hobbies-diy) in Educational Skills, 3D interactive guides to learn how to tie any knot. 

- Starred [Google AI Mode](https://workwatch.net/ai#specialized-chatbots) in Specialized Chatbots, AI Search Engine, that gives unlimited Gemini 3.

- Starred [Net4People](https://workwatch.net/privacy#anti-censorship) in Censorship Bypass section, worldwide censorship circumvention discussion, chats take place within the GitHub issue section. 

- Starred [NetBird](https://workwatch.net/privacy#vpn-server) in VPN Servers, WireGuard Mesh similar to Tailscale, except its open-source, self-hostable, and has a web UI. 

- Starred two [FreshRSS](https://workwatch.net/internet-tools#rss-readers) instances in RSS Readers (rss.workwatch + Prodottoinrete), its customizable, web-based, and allows unlimited feeds.
