#### YTS / Yify Note

YTS / Yify has many fake copycat sites out there, make sure you're on one of the official domains before downloading anything. To be extra protected from fake sites, check out [WORKWATCH SafeGuard](https://github.com/workwatch/WORKWATCH-SafeGuard) and the [WORKWATCH Filterlist](https://github.com/workwatch/WORKWATCHFilterlist).
